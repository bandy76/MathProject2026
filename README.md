数学分析动态可视化学习平台使用说明
这是一个集数学知识讲解、公式展示、动态动画演示、AI智能答疑于一体的数学分析交互式学习工具，专注多元微分、多元积分、无穷级数三大核心模块，支持电脑/平板/手机全端适配。
一、快速启动
1. 打开页面后，等待启动界面加载完成
2. 点击中央按钮 进入数学宇宙，即可进入主学习系统
3. 系统自动加载欢迎页，可直接点击章节卡片开始学习
二、界面布局与功能
1. 左侧边栏 · 章节导航
三大知识模块：多元微分学 / 多元积分学 / 无穷级数
点击模块标题可展开/折叠子菜单
点击章节按钮切换内容，蓝色高亮为当前章节
支持章节：Rⁿ空间、极限与连续、导数与微分、泰勒公式与极值、二重积分、三重积分、傅立叶级数
2. 中间区域 · 学习主界面
 章节头：分类标签 + 标题 + 核心概述
 核心公式栏：本章节最重要公式，支持LaTeX渲染
 知识点网格：4个卡片，讲解核心理解、例子、判断方法
 动态演示区：交互式动画 + 参数控制面板，直观理解几何意义
3. 右侧边栏 · AI提问助手
实时显示当前学习章节，AI自动关联上下文
状态指示灯：
绿色：DeepSeek AI已连接
灰色：本地代理未启动
红色：AI异常，启用本地答疑
快捷提问：核心公式、例题、动画含义，一键获取答案
自由提问：输入数学问题，AI给出精准解答与步骤
支持公式、概念辨析、解题思路、动画解释
三、动态演示使用方法
1. Rⁿ空间：调节维数n，观察高维向量投影与范数变化
2. 极限与连续：控制趋近速度与ε圆盘，查看多路径逼近效果
3. 导数与微分：调节曲率与切点位置，观察曲面、切平面、梯度
4. 泰勒公式：调整多项式次数与观察范围，对比逼近效果
5. 二重积分：改变网格密度，查看黎曼小柱逼近体积过程
6. 三重积分：调节球体半径与切片高度，计算切片面积与球体体积
7. 傅立叶级数：增减项数，观察方波逼近与吉布斯现象
四、AI助手使用指南
1. 快捷提问（推荐）
直接点击下方三个按钮：
核心公式：获取当前章节必背公式与含义
例题：获取典型例题与详细解题步骤
动画含义：看懂动画在演示什么数学原理
 
2. 自由提问
在输入框输入问题，例如：
多元极限怎么判断不存在？
二重积分极坐标替换怎么用？
梯度和方向导数有什么关系？
泰勒公式和傅立叶级数的区别？
AI会结合当前章节给出精准、简洁、带公式的回答。
